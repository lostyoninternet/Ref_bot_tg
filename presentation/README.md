# Презентация реферальной программы «Алабуга Политех» 2026

## Файлы

- **REFERRAL_PROGRAM_2026.md** — слайды в формате [Marp](https://marp.app/).

## Как открыть и экспортировать

### Вариант 1: VS Code + Marp

1. Установи расширение [Marp for VS Code](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode).
2. Открой `REFERRAL_PROGRAM_2026.md`.
3. Нажми «Open Preview to the Side» или команду `Marp: Open Preview`.
4. Экспорт в PDF: команда `Marp: Export Slide Deck` → выбери PDF (или HTML).

### Вариант 2: Marp CLI

```bash
npm install -g @marp-team/marp-cli
marp REFERRAL_PROGRAM_2026.md --pdf
# или
marp REFERRAL_PROGRAM_2026.md --html
```

### Вариант 3: Без Marp

Содержимое `REFERRAL_PROGRAM_2026.md` можно скопировать в Google Slides, PowerPoint или другой редактор и оформить слайды вручную (разделитель слайдов — `---`).

## Структура слайдов

1. Титул  
2. Цель и доп. цель (снижение стоимости лида)  
3. Способ (реферальная программа)  
4. Аудитория (абитуриенты, колледжисты, сотрудники)  
5. Механика: путь участника (3 шага)  
6. Система мотивации: грейды  
7. Таблица грейдов и призов (0–6, баллы, призы)  
8. Что входит в продукт (бот, админка, CRM)  
9. Безопасность и приватность  
10. Итог
